Oaxaca.py

Oaxaca-Blinder is a statistical method commonly used in labor and health economics. It is useful as it can help show what is explained and unexplained in differences between means for two groups, such as employment or wages. There are two different types of Oaxaca-Blinder, two-fold and three-fold. While similar, three-fold quantifies the interaction between the two effects in two-fold, which is left unshown in two-fold. However, two-fold is much easier to quantify than three-fold. In my GitHub repo, there is a good paper proposing the oaxaca command for STATA, which is where I got a lot of inspiration for design choices.

This package was started because, when studying this method for my Labor Economics class, I found there was no Python implementation. There was a group of students who said they had created one, but their GitHub repo has nothing in it. It also tested my ability to create a package that the entire world can see, like documenting your code and making it straightforward.
